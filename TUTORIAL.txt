BEM-VINDO(A) AO OMNIFETCH ULTIMATE v3.0!
Você acaba de adquirir um arsenal de engenharia de mídia de alto desempenho. 
Diferente de sites lotados de vírus e anúncios, o OmniFetch roda localmente, 
garantindo 100%% de privacidade, velocidade máxima e qualidade sem perdas.

=======================================================================================
[ 🚀 PASSO 01: O BOOTSTRAP (CONFIGURAÇÃO) ]
=======================================================================================

Antes dos motores rugirem, precisamos calibrar o sistema. Execute isso APENAS UMA VEZ.

1. Localize o arquivo "1 SETUP.bat".
2. Clique com o botão direito e selecione "EXECUTAR COMO ADMINISTRADOR".
3. O script irá baixar os motores portáteis (yt-dlp e FFmpeg) para a pasta 'motores'.
4. Ele também tentará instalar o Calibre (para e-books) silenciosamente via Winget.
5. Quando ler "🎉 MOTORES PORTÁTEIS INSTALADOS!", você está pronto para o combate.

=======================================================================================
[ 📡 PASSO 02: O BAIXADOR (2_DOWLOADER.bat) ]
=======================================================================================

O coração do arsenal. Baixa de YouTube, Instagram, TikTok, Drive e +1000 sites.

1. COPIE o link (URL) do conteúdo desejado.
2. ABRA o "2_DOWLOADER.bat".
3. COLE o link (Botão direito ou Ctrl+V) e aperte ENTER.
4. SELECIONE A CATEGORIA (Letras A a J).
5. SELECIONE O FORMATO (Números 1 a 100).
6. DESTINO: Aperte ENTER para salvar na pasta 'Downloads' ou digite um caminho manual.

DICA: Se colar um link de PLAYLIST, o sistema ativará o modo radar e perguntará se 
você quer o vídeo único ou a lista completa.

=======================================================================================
[ ⚡ PASSO 03: O CONVERSOR (3_CONVERSOR.bat) ]
=======================================================================================

Transforme qualquer arquivo local em formatos profissionais (Vídeo, Áudio, Foto, Docs).

1. ABRA o "3_CONVERSOR.bat".
2. COLE O CAMINHO do arquivo. (Dica: Clique no arquivo com Shift + Botão Direito 
   e escolha "Copiar como caminho").
3. ESCOLHA o que deseja (ex: Opção 1 para Vídeo, depois 101 para MP4 Universal).
4. DESTINO: Aperte ENTER para salvar na pasta 'Downloads'.

=======================================================================================
[ 🛠️ GUIA DE TROUBLESHOOTING (SOLUÇÃO DE ERROS) ]
=======================================================================================

❌ ERRO: "O Windows não pode encontrar yt-dlp ou ffmpeg"
✅ SOLUÇÃO: Você pulou o PASSO 01. Execute o "1 SETUP.bat" como Administrador.

❌ ERRO: "Acesso Negado ao criar pastas"
✅ SOLUÇÃO: Certifique-se de que a pasta do OmniFetch não está em um local restrito 
   (como C:\Windows). Tente mover a pasta OmniFetch para sua Área de Trabalho ou Documentos.

❌ ERRO: "Download Falhou (HTTP Error 403/404)"
✅ SOLUÇÃO: Provavelmente o site de origem bloqueou o acesso temporário. 
   Tente atualizar o yt-dlp rodando o SETUP novamente ou aguarde alguns minutos.

❌ ERRO: "Conversão de E-book não funciona"
✅ SOLUÇÃO: O motor Calibre precisa estar instalado no sistema. O SETUP tenta fazer 
   isso, mas se falhar, baixe o 'Calibre' manualmente no site oficial.
